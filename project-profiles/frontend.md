# Frontend Review Profile

Focus on:

- State consistency and stale closures.
- Effect dependencies and cleanup.
- Loading, error, and empty states.
- Accessibility of changed controls.
- Responsive layout and text overflow.
- API error handling and retry behavior.
- Unnecessary rendering in hot paths.

Avoid:

- Cosmetic comments that do not affect usability, accessibility, or maintainability.
- Rewriting component structure without a clear risk.
