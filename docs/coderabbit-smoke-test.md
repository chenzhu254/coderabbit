# CodeRabbit Smoke Test

This file exists to create a harmless pull request that verifies CodeRabbit can review this repository.

Expected result:

- GitHub receives a pull request from `chore/coderabbit-smoke-test`.
- CodeRabbit posts a review or walkthrough comment on the pull request.
- The repository uses `.coderabbit.yaml` from this repo root.

Configuration verification:

- Comment `@coderabbitai configuration` on the pull request.
- Confirm the response reports the active repository configuration from the root `.coderabbit.yaml`.
